# LINE 自動化模板截取指南

## 重要提醒
為避免數字變化導致模板匹配失敗，請按照以下指南正確截取模板圖片。

## 箭頭校正模板（favorite.png, friend.png, group.png, community.png）

### ❌ 錯誤做法
不要截取包含數字的完整區域，例如：
```
[收藏圖標] 收藏 (123)  [↓箭頭]
```
因為數字 `(123)` 會隨時變化，導致匹配失敗。

### ✅ 正確做法
**方法1：只截取左側固定部分**
只截取圖標和文字，不包含數字：
```
[收藏圖標] 收藏
```

**方法2：只截取右側箭頭（推薦）**
只截取右側的展開/收合箭頭：
```
[↓箭頭]
```
或
```
[→箭頭]
```

**方法3：分開截取（最穩定）**
- 截取左側圖標文字為一個模板
- 截取右側箭頭為另一個模板
- 程式會分別偵測後組合結果

### 截取步驟
1. 打開 LINE 應用程式
2. 找到左側聊天列表的分類欄位（收藏、好友、社群、群組）
3. 使用截圖工具（例如 Windows 的截圖工具、Snipping Tool）
4. **只截取右側的箭頭圖標**（約 20x20 到 30x30 像素）
5. 儲存為對應的檔名：
   - `favorite.png` - 收藏的箭頭
   - `friend.png` - 好友的箭頭
   - `group.png` - 群組的箭頭
   - `community.png` - 社群的箭頭

## 箭頭狀態模板（hide.png, show.png）

### hide.png（收合狀態的箭頭）
截取指向右邊的箭頭 `→`

### show.png（展開狀態的箭頭）
截取指向下邊的箭頭 `↓`

## 其他模板

### friend-list.png
截取「好友」分類的完整按鈕區域（用於開啟好友選單）

### greenchat.png
截取綠色聊天框圖示（表示有未讀訊息）

### message_cube.png
截取訊息立方體圖示

## 建議規格
- **格式**：PNG（支援透明背景）
- **尺寸**：
  - 箭頭：20x20 到 30x30 像素
  - 文字圖標：50x30 到 100x40 像素
- **背景**：透明或與 LINE 介面背景一致
- **解析度**：與你的螢幕實際顯示一致

## 測試方法
截取模板後，可以使用「測試」頁面中的「箭頭校正」功能來驗證模板是否正確。

## 疑難排解

### 問題：模板偵測失敗
**可能原因**：
1. 模板包含了會變化的數字
2. 螢幕縮放比例不同
3. LINE 版本更新導致介面改變

**解決方法**：
1. 重新截取模板，只包含固定不變的部分（箭頭）
2. 確認螢幕縮放比例為 100%
3. 在實際使用的電腦上截取模板

### 問題：數字變化後失敗
**解決方法**：
只截取右側箭頭，不要包含左側文字和數字部分。

## 範例

假設 LINE 介面顯示：
```
[★圖標] 收藏 (5)  [↓]
[👤圖標] 好友 (123) [↓]
[👥圖標] 群組 (45)  [→]
[🏢圖標] 社群 (12)  [→]
```

應該截取：
- `favorite.png` → 只截取最右邊的 `[↓]`
- `friend.png` → 只截取最右邊的 `[↓]`
- `group.png` → 只截取最右邊的 `[→]`
- `community.png` → 只截取最右邊的 `[→]`

這樣無論數字如何變化，都能穩定匹配。
