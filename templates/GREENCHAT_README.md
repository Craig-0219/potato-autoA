# greenchat.png 模板說明

## 用途
`greenchat.png` 用於依序開啟聊天窗測試中，檢測哪些好友有未讀訊息（綠色聊天框）。

## 如何創建
1. 在 LINE 好友列表中找到一個有未讀訊息的好友（會顯示綠色聊天框圖示）
2. 對該綠色聊天框圖示進行截圖
3. 將截圖保存為 `greenchat.png`
4. 放在 `templates/` 目錄下

## 建議規格
- 格式：PNG
- 尺寸：建議 30x30 到 50x50 像素（只截取綠色聊天框圖示部分）
- 背景：透明或與 LINE 好友列表背景一致

## 檢測說明
程式會使用此模板來識別好友列表中的綠色聊天框，並自動點擊開啟這些好友的聊天視窗。
